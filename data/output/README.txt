The output from the programs have been in this dirctory.

The directory syntactic contains the output from question_generator_syntactic. The question/answer pairs are marked with a [Y] or [N] for valid and invalid entries respectively.

The directory transformer contains the output from question_generator_transformerc. The question/answer pairs are marked with a [Y] or [N] for valid and invalid entries respectively. The files are named q<number>.txt where <number> is the number of questions requested.
